# Pokedex frontend project

## Development stack
- Next
- TypeScript
- Zustand
- Tailwind
- Styled-Components
- Twin

## Testing stack
- Jest
- React Testing Library
- Cypress

## Run project
- Install dependencies: `npm i`
- Run project: `npm run dev`
